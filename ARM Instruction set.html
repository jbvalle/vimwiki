ARM Instructions

Structure

MNEMONICS{S}{condition} {Rd}, Rn, Rm

  • MNEMONICS: Name of the instruction
  • {S}: an optional suffix. If S is specified the condition flags are updated on the result of the operation
  • {condition}: condition that is needed in order for the isntruction to be executed

Rm - flexible

  • Rm or second operand is called flexible because we can use various forms:
    #123 Immidiate value
    Rx Register x ie r1, r2 , etc
    Rx, ASR n Reg x with arythmetic shift right by n bits n = 1 ... 32
    Rx, LSL n Reg x with logical shift left by n bits n = 0 ... 31
    Rx, LSR n Reg x with logical shift right by n bits n = 1 ... 32
    Rx, ROR n Reg x with rotation right by n bits n = 1 ... 31
    Rx, RRX with Reg x with rotate by one bit with extend

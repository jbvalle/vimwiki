ARM GCC Inline Assembly

  • arm-gcc-compiler:
    • Format:
      • __asm volatile(code: output operand list: input operand list: clobber list);
      • volatile: used to tell the compiler not to apply any optimiations
      • code: Assembly mnemonic
      • output operand list: list of output operand list, seperated by commas
      • input operand list: list of input operand list, seperated by commas
      • clobber list: used to tell the compiler about modifications done by the assembler code

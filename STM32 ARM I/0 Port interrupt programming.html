STM32 ARM I/0 Port interrupt programming

  • I/o Pins interrupts first come through external interrupt EXTIx
  • All Pins of each Port with the same number are connected with a external interrupt line with the corresponding number:
    • All Px0 are connected with EXTI0
    • All Px1 are connected with EXTI1
    • etc.

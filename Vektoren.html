Vektoren

Terminologien

  • Man betrachtet n-Tupel Vektoren, n ist die anzahl an Komponenten des Vektors
  • Ein Ortsvektor ist ein Vektor ausgehend vom Urpsung O.

Operationen

Addition

Vorraussetzung für die Vektoraddition: v1 + v2 | rank(v1) = rank(v2)

v1 + v2 = (v1(1) + v1(2) + ... + v1(n)) + (v2(1) + v2(2) + ... + v2(n))

Skalar Multiplikation

k * v | k element R , v element R^n k * v = k * (v1,v2,...,vn)

Vektoreigenschaften im Vectorraum

Nullvektor

0 = [0,...,n-mal]

Vektoren Gleichheit

v1, v2 element R^n v1 = v2 | wenn v1 und v2 komponentenweise gleich sind.

Abstand zwischen zwei Punkten A , B

vector(A-to-B) = [B(1) - A(1), B(2) - A(2)] = [Bx - Ax; By - Ay, ...]

Norm eines Vektors

  • Für a element R^n

||a|| = sqrt(a(1)² + a(2)² + a(3)² + ... );

  • Für a element C^n

||a|| = sqrt(sum(|a_j|²) mit |a_j|² = a_j * compl.conjug(a_j) -> Für komplexe zahlen wird der Betrag genommen, da das Quadrat immer noch eine negative Zahl ergeben könnte, für die Länge aber ein positive Betrag nur sinnvoll ist.

  • Definition des Längenbegriffs/Norm eines Vektors im Rn oder Cn Sei V element Rn oder Cn mit einer Funktion || || : V -> [0,infinity)
    1. || a || > 0 , mit a != 0 Positivität
    2. ||k * a|| = |k| * ||a|| Homogenität
    3. ||a + b|| <= ||a|| + ||b|| Dreieckungleichung

mit a,b element V und k element Rn oder Cn -> normierter Raum

Unix Union

The general structure of the Unix Union describes the architecture of a Unix Operating system

  1. Layer: Kernel:
    • The Kernel is next software layer surrounding the hardware. It hold the highest priviliges and controls the inner working of the hardware
  2. Layer: Shell:
    • The shell is the interfacing layer communicating with the kernel over syscalls
    • It consists of a set of tools that enable communication with the kernel
  3. Layer Application:
    • This layer contains the actual program a user would interact with
    • This programs access the kernel by interfacing with the shell program by using the system calls provided
    • Using this approach makes a program more portable

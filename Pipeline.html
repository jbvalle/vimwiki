Pipe Line

  • Allows Processor to execute instructions every cycle
  • Every Cycle consists of multiple stages such as:
    • Fetch
    • Decode
    • Execute
  • In Every cycle an instruction enters the fetching stage, then is decoded. At the same time the current instruction is decoded the next instruction is fed into te fetching stage.
  • In The next cycle the first instruction is then executed while the previous one is beeing decoded, while at the same time another instruction is fed into the fetching phase

ARM - Cortex M - Registers

  • The ARM - CORTEX M Family is a subset of the ARM Instruction set
  • However their programming models are very similar to earlier versions such as arm7tdmi
  • In fact, instruction and registers concerning the first 15 Registers are the same

Specs

17 General Purpose Registers
1 Status Register
3 interrupt mask registers

Registers

  • Available : R15 : R0
    R13 Stack Pointer SP
    R14 Link Register LR
    R15 Program Counter PC
  • Special Registers:
    • xPSR
    • PRIMAS
    • FAULTMASK
    • BASEPRI
    • CONTROL
  • Stackpointer has:
    • Main Stack Pointer
    • Process Stack Pointer

Accessing Special Registers

  • For Addressing Special Registers we need:
    • MRS ; Read special Register
    • MSR ; Write into special register

xPSR

  • Can be accessed with following registers, which each are specialized registers for accessing special functions
    • APSR - Application PSR
    • EPSR - Execution PSR
    • IPSR -ISR

APSR

N 31 Null
Z 30 Zero
C 29 Carry
V 28 Overflow
Q 27 Sticky Flags, can only be declared by writing 0 to it
GE[3:0] 19 : 16 Greater than or equalt to flag

EPSR

ICI / IT 26 : 25 Interrupt Continuation Instruction bits / IF-THEN BIT
T 24 Thumb Bit
ICI / IT 15 : 8 Interrupt Continuation Instruction bits / IF-THEN BIT

IPSR

ISRNUM 7 : 0 Exception Handler for handling Faults

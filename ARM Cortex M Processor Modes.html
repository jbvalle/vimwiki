ARM Cortex Processort Modes

2 Stacks in ARM Cortex

  • R13 SP is used to point to RAM
  • Two Stack pointer registers are used:
    • PSP - Processor Stack pointer - used in thread mode
    • MSP - Main Stack Pointer - used in handler mode

Configure Stack Pointer wit Special function register CONTROL

  • NPRIV:
    • 0: Priviliged
    • 1: Unpriviliged
  • Active Stack Pointer ASP:
    • 0 MSP
    • 1 PSP
  • Flaoting Point context active FPCA:
    • 0 No floating point context active
    • 1 Floating point context active

Special Function Register

  • General Purpose Reigsers are R0 - R15 which can be accessed with MOV, LDR and STR
  • As for SFR these have to be move to General Purpose Register using the Main Stack pointer, manipulate them and then move the back using the Main Stack pointer

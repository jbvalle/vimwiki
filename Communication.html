Here the kernel handles communication between processes.

Two models are used for inter-process communication:

  • Message Passing: Meaning processes exchange messages with one another
  • Shared Memory: Processes share a memory region in order to communicate

Systemcalls that are sent are: pipe(), shmget(), mmap()

  • pipe():
    • This syscall is used to enable inter-process communication between processes
    • pipe() is used to open file-descriptors
  • shmget():
    • stands for shared memory segment
    • this syscall is used for shared memory communication
    • this syscall is sent in order to access the shared memory and thereby access the messages within the memory in order for processes to communicate
  • mmap():
    • This syscall is used to map or unmap files or devices into the content of virtual memory space of the processes

Memory

Performance - Memory - Size

  • A general rule is, the higher the performance the lesser the memory size
  • Cache is the fastest memory type

Memory Width

  • This is the number of memory returned per access
  • ie. a 16 bit memory needs two load fetch cycles for a 32. Bit data, however this is considered cheaper
  • ie. given we use the 16 Bit thumb mode with a memory width of 8 Bit, we would need 2 Cycles to fetch the complete 16 Bit data

Memory Types

  • ROM - Read only cannot be reprogrammed
  • Flash ROM:
    • usually used for devices firmware
    • can be written and read to
  • DRAM - Dynamic Random Access Memory
  • SRAM - static random acces memory:
    • faster than dram
  • SDRAM - Synchronous dynamic random access memory
    • faster than sram
  • EEPROM - can be written and read to
    • not all arm chip have an eeprom

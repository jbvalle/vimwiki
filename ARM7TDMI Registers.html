ARM7 Registers

Processor

  • 30 General Purpose Registers
  • 6 Status Registers
  • 1 Program Counter Register

Registers

  • Accessable Registers are different depending on the current mode
  • Overview:
R0 - R12 General Purpose Registers contain data or addresses
R13 Also known es Stack Pointer SP - points at the top element of the stack
R14 Linkder Register LR - used to store return location for functions
R15 Program Counter PC - it is readable and writable

CPSR - Current Program Status Register

N Z C V ------------- I F T M4 M3 M2 M1 M0
  • N, Z, C, V - Used for Status Registers
  • I | F - Used to set Normal Interrupt or Fast interrupt
  • T - Status Bit -> gives information about the status of the system
  • M4:M0 - is used to set processor mode xPSR[4:0]

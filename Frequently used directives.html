Frequently used directives

Thumb placed at the top to specify that the code is generated with thumb instrutions
CODE indicates the section for machine instructions - ROM
DATA indicates the section for global variables - RAM
AREA instructs the assembler to assemble a new code or data section
SPACE Reserves a block of memory and fills it with zeros
EXPORT makes an obect accessible from another file
GLOBAL same as EXPRT
IMPORT to acces an exported object
END places at tge end of each File
DCB , DCW, DCD places a byte, half word and word sized constand into memory (8, 16, 32)
EQU gives a symbolic name to a numeric constant

Operational Modes of the processor

These applies for M0/M3/M4

  • The processor can run in two modes:
    • Thread-mode
    • Handler-mode

Thread mode is also called User-mode. This mode is always entered by default in the processor's starting phase. Whenever the core meets an exception or external interrupt, the core changes to handler mode in order to run the Interrupt Service Routine (ISR) associated with that system exception or interrupt.

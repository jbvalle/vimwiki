Core Registers

Registers Function
R0 - r12 General Purpose Registers
Stack Pointer (SP - R13) Stores the address of last executed instruction
Link Register (LR - R14) Stores return information for subroutines, function calls and exceptions, on Reset -> Processor sets LR to 0xFFFFFFFF
Program Counter (PC - R15) Holds the address of the next instruction to be executed

Program Counter

  • Application Program Status Reigster APSR:
    • Hold N, Z, C, V, Q Flags
  • Interrupt Program Status Register IPSR:
    • Hold the current exception number
  • Execution Program Status Regiser EPSR:
    • Holds ICI/IT
    • Holds T Bit: if T=1 Thumb mode | T=0 ARM Mode

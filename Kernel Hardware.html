Kernel Hardware

  • Kernel Hardware are generally interfaced by so called devices
  • These devices can be either real or virtual
  • All devices have dedicated files in the /dev directory
  • These files are used for communication
  • Mandatory devices are hardware which must be present for the unix system to work like CPU:
    • These are invisible from the userspace
  • Peripheral devices are optional devices which can be accessed and are visible in the user space
  • types of devices:
    • char devices: These devices deliver and consume data byte by byte
    • block devices: These deliver and consume more than one byte of data

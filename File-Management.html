This Systemcall tells the kernel to create, read and write to a file (open, read,write, close):

  • open():
    • This Systemcall, call instructs to open a file in order to perform operations such as reading and writing
  • read():
    • This Systemcall instructs to open a file in reading mode
    • This Call can be executed by multiple processes on the same time for the same file.
  • write():
    • Open a file in writing mode
    • This systemcall make it possible to write to a file
    • Multiple Processes can NOT execute write() on the same file simltaneously
  • close():
    • Closes the opened file

Lineare Unabhängigkeit und Basis

Definitionen

Linearkombination

Die Summe der Vektoren a_i element einer Vektormenge V jeweils um k_i element eines Körpers skaliert ist eine Linear Kombination

Mittels der Linearkombination kann man z.b. im Rn mittel k_i * a_i + k_(i+1) * a_(i+1) + ... jeden Vektor im Rn erreichen. Dafür müssen die Vektoren aber Linear unabhängig sein, sprich keine Vielfache voneinander sein.

Lineare Unabhängigkeit

Diese ist dann erreich wenn die Linear Kombination von Vektoren die Null ergeben, nur dann erreicht ist, wenn die Skalierfaktoren k_i element R^n alle gleich 0 sind. Wenn es unterschiedliche Lösungen für k_i gibt dann sind die Vektoren linear abhängig.

Basisvektoren

Als Basis Vektoren werden die minimale Anzahl an linear unabhängigen Vektoren bezeichnet aus denen man im Rn jeden Vektor als lineare Kombination aus den Basisvektoren bilden kann. Die minimale Menge dieser Basisvektoren die jeden Vektor im Rn erzeugen können nemmt man auch minimales Erzeugendensystem.

Dimension

Als Dimension oder dim(V) bezeichnet man die maximale Anzahl von linear unabhängigen Vektoren in einem Vektorraum. Man unterscheidet dabei zwischen endlich dimensional Vektorraum und unendlichdimensional Vektorraum.

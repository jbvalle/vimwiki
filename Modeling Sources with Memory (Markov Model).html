Markov Model

Modeling Statistical Dependencies

  • These are described using conditional probabilities
  • Memory:
    • if Statistical Dependencies exist
  • Memoryless:
    • if there is no Statistical Dependencies
  • Conditional Probablity:
    • These are used to describe the probability of one symbol succeding a nother
    • P[s(t+1) = s1 | s(t) = s2] => s2 predecessor and s1 successor = P[s1|s2]
  • Joint Probability:
    • These are used to describe the probability of generally observing the succeding and preceding of favoured elements
    • P[s(t) = s1, s(t+1) = s2] = P[s2|s1] * P[s1] => s1 successor and s2 predecessor

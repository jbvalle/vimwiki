• Create following files:
  • File: mymodule.c =
  • File: Makefile =
  • Execute the Makefile
$ make
  • By now the kernel object file should have been created, now load the module to the kernel:
$ sudo insmod mymodule.ko
  • Inspect the modules with:
$ lsmod | grep mymodule
  • Now check for the kernel log with:
$ dmesg | tail -5
  • Now remove the module with:
$ sudo rm mymodule

1. Create following files:
  • File: mymodule.c =
  • File: Makefile =
  1. Execute the Makefile
$ make
  1. By now the kernel object file should have been created, now load the module to the kernel:
$ sudo insmod mymodule.ko
  1. Inspect the modules with:
$ lsmod | grep mymodule
  1. Now check for the kernel log with:
$ dmesg | tail -5
  1. Now remove the module with:
$ sudo rm mymodule

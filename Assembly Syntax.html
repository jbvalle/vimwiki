Assembly Syntax

4 Fields

| 1. Label | 2. OPCODE | 3. OPERAND | ; COMMENT

  • Label:
    • Optional - Used to find the position of current instruction in memory
  • Opcode:
    • Specifies the processor operation
  • Operand:
    • Specifies source / destination of data to be processed
  • Comment:
    • optional - used to explain code meaning

Example

Label Opcode Operand Comment
store_routine STR R0 , [R1] ; Store the value of R0 in R1

ARM7TDMI Vector Table

  • The Vector Table is a collection of addresses the program counter can jump to in memory, where it continues executing the instructions in starting that address
  • ie. in the case of an interrupt the program counter will jump to the address associated with that interrupt

ARM7TDI Specific Vector Table

Exception Mode Vector Address
Reset SVC 0x0000 0000
Undef. instruction UNDEF 0x0000 0004
Software interrupt (SVC) SVC 0x0000 0008
Prefetch Abort ABORT 0x0000 000C
Data Abort ABORT 0x0000 0010
IRQ IRQ 0x0000 0018
FIQ FIQ 0x0000 001C

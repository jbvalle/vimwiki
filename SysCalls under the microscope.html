SysCalls under the microscope

  1. A Userprocess make a function call to read a file
  2. The Userprocess invokes a function which is executed as defined in libc
  3. The function in Libc then invokes a system call which is redirected to the kernel
  4. The kernel executes the system call and access the filesystem to read the file
  5. The file is then copied to a buffer inside the userspace
  6. Once the kernel process is done the user process continues, until then is stays frozen

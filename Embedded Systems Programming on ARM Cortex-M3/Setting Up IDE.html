Setting Up IDE

  1. New STM32 Project

Compilation

  1. Right Click Parent Directory -> Build Project

Remove FPU Warning

  • FPU : Floating Point Unit
  • The FPU has to be disabled to remove the warning:
    • Right click Parent Directory -> Properties: C/C++ Build -> Setting -> Tool Settings -> MCU Settings: Floating-point unit: -> Set to none | Floating Point ABI -> set to SW implementation

Flashing Code unto the Target

  • Board is put into Debug mode -> Configure Debug Mode first:
    • Debug as -> Debugger -> Debug probe: STM-Link(ST-LINK GDB Server) | Interface : SWD | SWV : Enable
  • Right Click Parent Directory -> Debug as -> STM32 Cortex-m C/C++ Application

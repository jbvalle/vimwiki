Load and Store

Address Mode: Offset - Using Immidiate value

str r2, [r1, #2]

  • store value stored in (r1 + 2) in r2 without changing value in r1

Address Mode: Offset - Using Immidiate value -> Pre Indexed

str r2, [r1, #2]!

  • store value stored in (r1 + 2) in r2, Register R1 changed to R1 = R1 + 2

Address Mode: Offset - Using Immidiate value -> Post Indexed

ldr r3, [r1], #4

  • load Value found in r1 into register r3, and change r1 = r1 + 4
